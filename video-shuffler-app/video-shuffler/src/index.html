yt-dlp Required

YouTube Sync needs yt-dlp and ffmpeg. Follow the steps for your system:

Mac:

  1. Install Homebrew (if not installed):
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Then install yt-dlp and ffmpeg:
    brew install yt-dlp ffmpeg

Windows:

  1. Download yt-dlp.exe from:
    https://github.com/yt-dlp/yt-dlp/releases/latest
    Save it to: C:\yt-dlp\yt-dlp.exe
  2. Download ffmpeg from:
    https://www.gyan.dev/ffmpeg/builds/
    Get "ffmpeg-release-essentials.zip", extract it,
    copy ffmpeg.exe into C:\yt-dlp\
  3. Add C:\yt-dlp to your system PATH:
    Search "Environment Variables" in Start → Edit the system environment variables → Environment Variables → select Path → Edit → New → type C:\yt-dlp → OK
